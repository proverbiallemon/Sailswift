What's New in v1.1.2

New Features

  • Action Bar - New bottom toolbar with quick access buttons for multi-select, toggle, delete, open folder, and refresh
  • Multi-Select Mode - Toggle button to enter selection mode without holding Command
  • Modpack to Profile - Option to save imported modpacks as profiles
  • Load Order Auto-Sync - All enabled mods now show their load order position consistently

Bug Fixes

  • Fixed folders not responding to clicks consistently
  • Individual mod file states now preserved in modpack export/import
  • Modpack export count now correctly shows folders instead of individual files
  • Load order numbers display for all enabled mods, not just those toggled in Sailswift

Changes

  • Toggle, Delete, Refresh, and Open Folder buttons moved from toolbar to action bar
